RIQOS PRODUCTION WEBSITE

Developer: Taariq Ismail

Qualification: Bachelor of Business Systems In Information Technology

Project Type: Multi-page Responsive Website

Languages Used: HTML5 & External CSS3



1. Project Overview
This project is a redesigned and improved version of my original Riqos Production website. The first version was built using HTML only. In this upgraded version, I implemented external CSS, responsive design principles, and improved layout structure to create a more professional and scalable website.
The purpose of the website is to represent Riqos Production as a structured creative label. The site introduces the brand, presents services, showcases artists, and allows potential clients to make professional inquiries.
The entire website was built using HTML5 and external CSS only. No JavaScript was used in order to demonstrate strong understanding of layout, styling, and responsiveness using CSS alone.



2. Website Structure
The website consists of five linked pages:

2.1 Home (index.html)
The home page introduces the brand identity and mission. It includes a hero section, brand description, and call-to-action elements.


2.2 About (about.html)
This page explains the background of Riqos Productions and includes a Spotify-style artist roster.
Each artist profile uses the <details> and <summary> elements to allow expandable content, improving user interaction while maintaining clean structure.

2.3 Services (services.html)
This page outlines the services offered:
Music Production
Audio Engineering
Music Video Production
Video Production
Artist Development
The page includes dedicated sections for embedded videos and structured content blocks.

2.4 Gallery (gallery.html)
The gallery page displays:
Four images
Three video elements
The layout uses CSS Grid and adjusts responsively depending on screen size.

2.5 Contact (contact.html)
The contact page includes a structured inquiry form and detailed contact information. The layout was designed to resemble a professional studio dashboard interface.





3. Design & Technical Implementation
The website demonstrates the following technical concepts:
External CSS linked across all pages
CSS Grid and Flexbox for layout control
Media queries for responsiveness
Navigation bar with wrapping behaviour
Use of clamp() for responsive font sizing
Object-fit for consistent image scaling
Hover transitions and visual feedback effects
The colour scheme uses a dark background with purple accents to reflect a modern music industry aesthetic.


4. Responsiveness
Responsive design was implemented using:
flex-wrap for navigation bar adaptability
CSS Grid with auto-fit for dynamic layouts
Media queries for mobile-specific behaviour
Horizontal swipe carousel effect on mobile screens
Scalable font sizes using clamp()
These techniques ensure usability across desktop and mobile devices.



5. Code Structure & Comments
Descriptive comments were added throughout both the HTML and CSS files.
The comments explain:
Section grouping
Layout decisions
Styling logic
Responsive adjustments
This improves readability, maintainability, and clarity of the development process.


6. Academic Alignment
This project demonstrates understanding of:
Semantic HTML structure
Separation of structure and styling
Responsive web design principles
User interface design fundamentals
Organised file structure and documentation


7. Conclusion
The upgraded version of the Riqos Productions website reflects a significant improvement from the initial HTML-only version. It demonstrates structured multi-page navigation, consistent external styling, responsive behaviour, and professional presentation.
The project meets the required web development standards for a first-year Bachelor of Information Technology student and reflects applied understanding of both design and technical implementation.
